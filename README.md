# MR.todo (todo管理くん)
GAS(Google Apps Script)を使った todo を管理してくれる 自作 Slack App  
<!-- TODO: できれば英語でも書く   -->  

## Description
GAS を使い、スプレッドシートで todo を管理します。  
Slack App としてSlackから入力を受け付けたり、通知をします。  

***DEMO:***  
![タスクの追加](./image/add_task.gif)  

## Features
- Slackからタスクを管理できます
    - タスクの追加  
    - タスクのリマインド  
- タスクはスプレッドシートで管理されます

###
タスクの追加
